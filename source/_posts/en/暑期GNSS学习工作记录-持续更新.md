## 07.02

- 用大模型快速解读一下生成的图片，下面分享一些我的理解：
  ![](demoFigure1.png)

  这个图是**原始信号概览**，最明显的是能看出来是否有异常的干扰与欺骗，其中四张图分别是I信号与Q信号的时域图和直方图，对于I信号与Q信号，大模型的理解如下：

> - **I和Q值是经过接收机前端处理后，用来“描述”原始卫星信号的一种数学表示方法。**I和Q值是经过接收机前端处理后，用来“描述”原始卫星信号的一种数学表示方法。
> - **I (同相) 分量**: 代表了信号中与接收机本地产生的**参考载波**“同频同相”的那部分能量。在GNSS中，我们设计的接收机就是要让**所有有用的数据信息**都承载在I分量上。
> - **Q (正交) 分量**: 代表了信号中与接收机本地产生的参考载波“同频但相位差90度”的那部分能量。在一个理想的、完美锁定的接收机中，Q分量应该**只包含噪声**，不包含任何有用信息。

​		时域图的锯齿状折线展现出了原始数据存在一定的噪声，而直方图展现出高斯分布符合中心极限定理给出的一般结果，说明没有异常的、恶意干扰的噪声。

​	![](demoFigure2.png)

​	图Figure2是**信号捕获结果**，每个小图对应着一个对接到的卫星，比较平坦的低值便是大量和本地码无法对接的噪声，而峰值便是对接成功的卫星信号。

​	![](demoFigure3.png)

​	Figure3便是展现了哪些卫星被成功捕获，值得我们进一步跟踪。这里貌似是用5-10的一个阈值进行截断的，~~mark一下，比较好奇不同阈值会有怎样的影响~~

​	![](demoFigure4.png)

​	纵轴`C/N0`，中文翻作“载噪比”，全称为**载波功率和噪声功率谱密度**，值越高，信号强度越高。然而在实际情境中，图像的上升或者下降都是异常事件，如果遇到**同步欺骗信号**，所有卫星对应的折线会**整体异常上升**；遇到**干扰、异步欺骗信号**则会**同步异常下跌**。

​	![](demoPRN4.png)

1. 最上面两幅图展现了**锁相环(PLL)**的工作效果非常好，将信号能量集中在I信号上，能量泄露非常少。
2. 中左展现了**锁频换(FLL)**快速修正频率误差的过程，在0.5秒后便趋于稳定、开始精确跟踪。
3. 中右展现了捕获信号的**整体强度**。
4. 下面两侧的**PLL**和**FLL**锁定图可以看到二者迅速上升到1，展现了二者成功锁定。中间的**延迟锁定环(DLL)**展现了**零均值噪声**，在0值附近抖动越小，码跟踪精度越高。

- 简单搭建了一个双向的CNN-LSTM模型，但还是过拟合了，训练了3个epoch便获得了100%的准确度。model核心代码放在下面👇。

```python
# --- 第1部分：CNN层 (用于提取空间特征) ---
# CNN层将每个时间点的 (卫星数S, 特征数N) "图像" 作为输入
self.cnn_layers = nn.Sequential(
    # 输入: (Batch, 1, S, N)
    nn.Conv2d(in_channels=1, out_channels=32, kernel_size=(3, 3), padding=1),
    nn.ReLU(),
    nn.BatchNorm2d(32),
    nn.MaxPool2d(kernel_size=(2, 1)), # 在卫星维度上做池化

    nn.Conv2d(in_channels=32, out_channels=64, kernel_size=(3, 3), padding=1),
    nn.ReLU(),
    nn.BatchNorm2d(64),
    nn.MaxPool2d(kernel_size=(2, 1))
)

# --- 计算CNN输出的维度，为LSTM做准备 ---
# 经过两次 (2, 1) 的池化，S维度会减半两次
cnn_output_height = S // 4
# N维度因为padding=1所以不变
cnn_output_width = N
self.lstm_input_size = 64 * cnn_output_height * cnn_output_width

# --- 第2部分：LSTM层 (用于学习时间序列特征) ---
self.lstm = nn.LSTM(
    input_size=self.lstm_input_size,
    hidden_size=128, # LSTM隐藏层的大小
    num_layers=2,    # LSTM的层数
    batch_first=True,# 输入数据的第一个维度是Batch
    dropout=0.5,     # 在多层LSTM之间添加dropout防止过拟合
    bidirectional=True # 使用双向LSTM
)

# --- 第3部分：全连接分类层 ---
# 因为是双向LSTM, 输出维度是 hidden_size * 2
self.classifier = nn.Sequential(
    nn.Linear(128 * 2, 64),
    nn.ReLU(),
    nn.Dropout(0.5),
    nn.Linear(64, num_classes)
)
```
